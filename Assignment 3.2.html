the CSS Box Model

This is a Home page, create a div box and set your large content inside a div.The CSS box model is essentially a box that wraps around every HTML element. It consists of: borders, padding, margins, and the actual content.

This text is the actual content of the box. We have added a 25px padding, 25px margin and a 25px green border. You can use the overflow property when you want to have better control of the layout. The overflow property specifies what happens if content overflows an element's box.
